**CommonGround**

*Planned Alerts*

v0.1  ·  March 2026

*This document defines the alert taxonomy for CommonGround. Each alert carries an explicit claim type, confidence level, and epistemic caution in accordance with the Guiding Principles Section V. Alerts are not conclusions — they are invitations to investigate and act.*

# Claim Type Reference

- CORRELATION — These conditions tend to co-occur. We do not know why. Use language: 'is associated with,' 'tends to precede,' 'co-occurs with.'

- MECHANISM — Correlation with a plausible causal pathway supported by external research. We can speak with more confidence but must name the research basis and local limitation.

- DOCUMENTED — Primary source evidence of a specific actor or event in this place at this time. Direct language permitted.

# Tier 1: Immediate Action

High-confidence signals requiring rapid community response. Reserved for documented events or strong mechanism-based patterns.

## Alert T1-H01: Mass Eviction Filing Event

| **Field** | **Detail** |
| --- | --- |
| Alert ID | T1-H01 |
| Claim type | DOCUMENTED |
| Severity tier | Tier 1 — Immediate |
| Primary sources | Wisconsin CCAP, Eviction Lab |
| Trigger condition | 5+ eviction filings by the same plaintiff within a 30-day window in a single building or block |

**What the data shows: **A specific landlord or property management company has filed multiple evictions in a concentrated area and time window. This pattern is associated with coordinated displacement strategy.

**Recommended action: **Connect affected tenants with legal aid (NLG, Legal Action of Wisconsin). Alert housing advocates. Document the plaintiff name for community awareness. Share Tenant Rights resources.

***Epistemic caution: ****This is a documented actor pattern, not a causal inference. The filings are real. The intent behind them is not established by the data alone.*

## Alert T1-E01: Severe Weather + High Vulnerability Overlap

| **Field** | **Detail** |
| --- | --- |
| Alert ID | T1-E01 |
| Claim type | MECHANISM |
| Severity tier | Tier 1 — Immediate |
| Primary sources | NWS, CDC SVI, HRSA |
| Trigger condition | NWS Tier 3+ weather alert issued for areas with CDC SVI score above 0.75 |

**What the data shows: **A severe weather event is forecast or occurring in a census tract with high social vulnerability. Research consistently shows that high-SVI populations face greater harm from the same weather event due to housing quality, access to transportation, and economic capacity to respond.

**Recommended action: **Alert mutual aid networks serving the affected tracts. Surface nearest warming/cooling centers and shelters. Activate the community resource layer for the affected area. Share NWS alert language directly.

***Epistemic caution: ****SVI predicts vulnerability at the aggregate level. Individuals in high-SVI tracts vary widely. This alert is a targeting tool for mutual aid, not a prediction about specific households.*

# Tier 2: Emerging Pattern

Cross-metric signals that suggest a developing situation. Require community attention but not immediate emergency response.

## Alert T2-H01: Compounding Displacement Pressure

| **Field** | **Detail** |
| --- | --- |
| Alert ID | T2-H01 |
| Claim type | CORRELATION |
| Severity tier | Tier 2 — Emerging |
| Primary sources | Eviction Lab, USDA SNAP, Green Bay permits, HUD CHAS |
| Trigger condition | Eviction filings up >10% quarter-over-quarter AND renovation permits up >15% year-over-year in the same zip code |

**What the data shows: **Two indicators associated with neighborhood displacement are co-occurring in the same area. Historically, this combination has preceded measurable displacement in comparable neighborhoods.

**Recommended action: **Alert housing advocates to increase outreach in the area. Flag for community organizing attention. Surface tenant rights resources proactively. Prepare documentation for potential public comment processes.

***Epistemic caution: ****This is a correlation, not a prediction. Both indicators can rise for reasons unrelated to coordinated displacement. Do not name specific landlords or properties unless CCAP plaintiff data independently supports it.*

# Tier 3: Situational Awareness

Informational alerts that build community awareness without requiring immediate action.

| Severity tier | Tier 3 — Informational |
| Primary sources | NWS, Green Bay permits, HRSA, AirNow, OSM |

**What the data shows: **A public assembly is permitted in the area. This briefing compiles publicly available safety-relevant information: current and forecast weather, AQI and wind direction, nearest FQHCs and urgent care, approved permit route geometry, and active mutual aid presence.

**Recommended action: **Share with event participants, medic teams, and legal observers. No action required — this is a compiled briefing of already-public information.

***Epistemic caution: ****All information in this briefing is public and symmetric — it is equally available to all parties. The briefing does not predict or recommend any specific action beyond awareness.*
